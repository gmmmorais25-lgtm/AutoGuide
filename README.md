# 🚗 AutoGuide

O **AutoGuide** é uma aplicação web desenvolvida para ajudar usuários a encontrar o carro ideal de forma simples, rápida e visual.

O sistema permite filtrar veículos por marca, categoria e busca por nome, além de visualizar informações detalhadas de cada carro.

---

## 🔥 Funcionalidades

- 🔍 Busca por nome ou marca
- 🏷️ Filtro por categoria (SUV, Sedan, Econômico, etc.)
- 🏢 Filtro por marca
- 📄 Modal com detalhes completos do carro
- 🌙 Modo Dark / Light
- 💻 Interface moderna e responsiva
- 🚗 Vários carros organizados em categorias

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
